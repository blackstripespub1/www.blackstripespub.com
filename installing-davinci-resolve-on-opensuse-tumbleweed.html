Installing DaVinci Resolve on OpenSUSE Tumbleweed

Darren Twa April 19, 2023 OpenSUSE Tumbleweed

Follow the basic instructions in the install PDF: make the installer executable and run it.

You will then be given a list of missing packages (if any). Install these.

If running the installer again gives an error, then it is looking for specific names that do not match the libraries you have just installed. To bypass this check (because you are certain you have installed the files) type:

SKIP_PACKAGE_CHECK=1 ./DaVinci_Resolve_18.5b1_Linux.run

Of course, if you need to have the executable above match your downloaded file.
